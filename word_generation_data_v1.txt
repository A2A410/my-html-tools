SECTION 10: ALGORITHMIC GENERATION RULES
================================================================================

RULE 1: Position-Based Selection
For each position N (1-12):
  - Query position favorability table (Section 4)
  - Weight candidates by position probability
  - Apply 3× penalty to letters already used 3 consecutive times

RULE 2: Bigram Filtering
For position N > 1:
  - Query bigram matrix for letter[N-1] → letter[N]
  - Eliminate transitions with 0% probability
  - Weight remaining by bigram frequency

RULE 3: Trigram Bonus
For position N > 2:
  - Check if letter[N-2,N-1,N] forms known trigram
  - Apply 1.5× multiplier to favorability if match found
  - Reference Section 3 for common trigrams

RULE 4: Consecutive Letter Constraint
- Track current consecutive count for each letter
- If count == 3: Set probability to 0% for 4th occurrence
- Reset counter when different letter selected

RULE 5: Phonotactic Validation
- Check against forbidden bigram lists (Section 7)
- If position == 1-2: Use initial bigram restrictions
- If position == N-1 to N (final): Use final bigram restrictions
- Invalid combinations receive 0% probability

RULE 6: Word Termination Decision
For position N ≥ 2:
  - Calculate termination probability from Section 8
  - Check if current sequence forms valid word ending
  - Apply length distribution weights (Section 6)
  - Terminate if: (random() < termination_probability) AND (valid_ending == true)

RULE 7: Normalization
After applying all filters:
  - Sum all non-zero probabilities
  - Normalize to 100%
  - Use weighted random selection for final choice

SECTION 11: PRACTICAL IMPLEMENTATION PSEUDOCODE
================================================================================

FUNCTION generate_word(max_length = 12):
    word = ""
    consecutive_count = {}
    
    FOR position FROM 1 TO max_length:
        candidates = []
        
        // Get base probabilities for this position
        base_probs = get_position_favorability(position)
        
        FOR each letter IN alphabet:
            prob = base_probs[letter]
            
            // Apply consecutive letter constraint
            IF consecutive_count[letter] >= 3:
                prob = 0
            END IF
            
            // Apply bigram filtering (if not first letter)
            IF position > 1:
                bigram_prob = get_bigram_probability(word[-1], letter)
                prob *= bigram_prob
            END IF
            
            // Apply trigram bonus (if applicable)
            IF position > 2:
                IF is_common_trigram(word[-2:], letter):
                    prob *= 1.5
                END IF
            END IF
            
            // Apply phonotactic rules
            IF NOT is_phonotactically_valid(word, letter, position):
                prob = 0
            END IF
            
            candidates.append({letter: prob})
        END FOR
        
        // Normalize probabilities
        total = sum(candidates.values())
        FOR each candidate:
            candidate.prob /= total
        END FOR
        
        // Select letter using weighted random
        selected = weighted_random_choice(candidates)
        word += selected
        
        // Update consecutive count
        IF word[-1] == selected:
            consecutive_count[selected] += 1
        ELSE:
            consecutive_count = {selected: 1}
        END IF
        
        // Check for word termination
        IF should_terminate(word, position):
            BREAK
        END IF
